# OpenBond

Connect OpenBond to Claude, Codex, and ChatGPT through JoAi's hosted MCP app server.

[Open app page](https://joai.ai/en/apps/openbond) · [Browse all apps](https://joai.ai/apps)

## Works With

- ChatGPT
- Claude
- Codex
- Cursor
- Other MCP clients

## Install

### Claude

```bash
claude plugin marketplace add --scope user JoAiHQ/claude-plugins
claude plugin install --scope user joai-openbond@joai-claude-plugins
```

### Codex

```bash
codex mcp add joai-openbond --url https://cortex.joai.ai/mcp/apps/openbond
```

### Cursor

Copy to local plugins:

```bash
git clone https://github.com/joai-plugins/openbond.git ~/.cursor/plugins/local/joai-openbond
```

Or use the [MCP deeplink](cursor://anysphere.cursor-deeplink/mcp/install?name=joai-openbond&config=eyJtY3BTZXJ2ZXJzIjp7ImpvYWktb3BlbmJvbmQiOnsidXJsIjoiaHR0cHM6Ly9jb3J0ZXguam9haS5haS9tY3AvYXBwcy9vcGVuYm9uZCJ9fX0=) to install directly.

### ChatGPT

Paste the MCP URL in ChatGPT settings:

```
https://cortex.joai.ai/mcp/apps/openbond
```
